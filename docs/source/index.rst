Social Interaction Cloud Documentation
======================================

.. image:: _static/sic_logo.svg
   :alt: SIC Logo
   :align: center
   :width: 300px

Welcome to the Social Interaction Cloud (SIC) framework documentation. This framework provides a comprehensive platform for building social interaction applications with various robotic platforms and AI services.

Overview
--------

The Social Interaction Cloud framework is designed to facilitate the development of social interaction applications by providing:

* **Core Framework**: Fundamental components for building modular applications
* **Device Support**: Interfaces for various robotic platforms (Pepper, NAO, AlphaMini, Desktop)
* **AI Services**: Integration with speech recognition, natural language processing, computer vision, and more
* **Message System**: Robust communication between components
* **Component Management**: Easy deployment and management of distributed components
